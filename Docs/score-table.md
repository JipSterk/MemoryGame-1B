# Score tabel

### Data

- 1 tile pair is worth 100 points

- board will have multiple layouts
    - 4 * 4 = 16 tiles
        - minimum score of 16 tiles * 100 points each 1600
    - 6 * 6 = 36 tiles
        - minimum score of 36 tiles * 100 points each 3600
